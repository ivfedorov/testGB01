# Инструкция для работы с Markdown
*__Markdown__* — язык текстовой разметки, созданный писателем и блогером Джоном Грубером. Он предназначен для создания красиво оформленных текстов.

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звездочками (*) или знаком нижнего подчеркивания (_). Например, *вот так*

Чтобы выделить текст полужирным необходимо обрамить его звездочками (**) или знаками нижнего подчеркивания (__). Например, __вот так__

Альтернативные способы необходимы, если есть совмещение. К примеру курсивный полужирный: __*вот так*__

## Работа с изображениями

Чтобы вставить изображение в текст, надо написать следующее: 

знак (<) img src = "название картинки" width = "ширина" height = "высота" (>)

## Списки

Чтобы добавить ненумерованнные списки необходимо пункты выделить (*) или (+). Например, вот так:
* Элемент 1
* Элемент 2
+ Элемент 3

Чтобы добавить нумерованнные списки необходимо пункты пронумеровать (1.) Например, вот так:
1. Элемент 1
2. Элемент 2
3. Элемент 3

## Ссылки
Существует два варианта оформления ссылок. Markdown поддерживает два стиля оформления ссылок:
+ Гиперссылка, с немедленным указанием адреса (внутритекстовая);
+ Гиперссылка, подобная сноске.

Подразумевается, что помимо URL-адреса существует еще текст ссылки. Он заключается в квадратные скобки. Для создания внутритекстовой гиперссылки необходимо использовать круглые скобки сразу после закрывающей квадратной. Внутри них необходимо поместить URL-адрес. В них же возможно расположить название, заключенное в кавычки, которое будет отображаться при наведении, но этот пункт не является обязательным.

Первый вариант — обычная вставка в текст. Вот так: [Шпаргалка Markdown](https://evileg.com/ru/pages/markdown/#Links)

Второй вариант — оформление ссылки в виде сноски. Вот так: https://evileg.com/ru/pages/markdown/#Links

## Цитаты
Для обозначения цитат в языке Markdown используется знак «больше» («>»). Его можно вставлять как перед каждой строкой цитаты, так и только перед первой строкой параграфа. Также синтаксис Markdown позволяет создавать вложенные цитаты (цитаты внутри цитат). Для их разметки используются дополнительные уровни знаков цитирования («>»). Цитаты в Markdown могут содержать всевозможные элементы разметки. 

>Это пример цитаты,
>в которой перед каждой строкой
>ставится угловая скобка.
 
>Это пример цитаты,
в которой угловая скобка
ставится только перед началом нового параграфа.


> Первый уровень цитирования
>> Второй уровень цитирования
>>> Третий уровень цитирования
>
>Первый уровень цитирования
